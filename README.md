### 🚀 Proyek Sorotan

*   **TDAS (Tongue Diagnostic Automatic System):** Sistem diagnosis otomatis berbasis klasifikasi citra medis (*Tongue Body*) menggunakan model *Transformer* mutakhir.
*   **Dia Predik:** Aplikasi *mobile* yang mengimplementasikan model klasifikasi riset untuk pengguna akhir (dibangun dengan Flutter).
*   **MOM Sync:** Aplikasi berbasis web (*Laravel* & *Tailwind CSS*) untuk manajemen *Minutes of Meeting* perusahaan secara efisien.
*   **Hydroponix:** Platform pembelajaran berbasis web yang interaktif untuk kursus hidroponik.
*   **Scotopia:** Proyek peluncuran produk dan *branding* untuk bisnis penjualan produk digital.
*   **Multimodal Emotion Detection:** Sistem pendeteksi emosi multimodal yang dibangun bersama tim untuk kompetisi *big data*.
